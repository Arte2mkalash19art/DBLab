## 1. Вербальна модель предметної області
Система призначена для автоматизації процесу продажу авіаквитків. Вона зберігає дані про країни, міста, типи літаків та конкретні рейси. Користувачі можуть купувати квитки на рейси, при цьому система автоматично розраховує фінальну вартість з урахуванням класу місця (Economy, Business, First) та додаткових зборів (паливний збір, страховка тощо).

## 2. Бізнес-правила
1. **Унікальність:** Кожен рейс (`FLIGHT`) повинен мати унікальний номер.
2. **Географія:** Місто вильоту не може збігатися з містом прибуття.
3. **Вікові обмеження:** Ціна квитка та збори не можуть бути від’ємними.
4. **Цілісність:** Видалення країни неможливе, якщо в ній зареєстровані міста (FK constraint).
